Web Service Security (WS-Security)
----------------------------------
WS-Security also describes how to encode binary security tokens and attach them to SOAP messages. 
Specifically, the WS-Security profile specifications describe how to encode the following tokens:
	1. Username tokens
	2. X.509 certificates
	3. SAML assertions
	4. Kerberos tickets
	5. LTPA binary tokens
